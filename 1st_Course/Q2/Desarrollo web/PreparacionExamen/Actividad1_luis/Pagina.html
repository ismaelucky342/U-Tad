Google Chrome
Con su lanzamiento en 2008 estableció un nuevo estandar de funcionalidad.
Por un lado, satisface al usuario con tiempos de carga reducidos, y por otro, ha perfeccionado la sincronización multiplataforma.
Caracteristicas:
-Apto para equipo de sobremesa, portátiles y dispositivos móviles
-Disponible para Windows, OS X, Linux, Android e IOS
-Tiempos de carga rápidos
-Muy seguros gracias a las actualizaciones regulares
-Amplia gama de plugins y extensiones
-Bloqueador de anuncios preinstalado contra la publicidad molesta

Mozilla Firefox
El navegador gratuito del proyecto Mozilla aparecería en 2002 bajo el nombre de Phoenix.
Su sucesor sería Mozilla Firebid y en 2004 se lanzaría la primera versión con el nombre de Firefox.
Las ventajas del sucesor directo de Netscape no solo incluyen la solidez de sus estándares de seguridad,
sino también una gran capacidad de expansión y compatibilidad con sistemas operativos como
Windows, Linux y OS X, Firefox también está disponible como aplicación móvil para dispositivos Android e IOS
Caracteristicas:
-Apto para equipos de sobremesa, portátiles y dispositivos móviles
-Disponible para Windows, macOS, Linux, Android e IOS
-Buena seguridad y excelente protección de datos
-El navegador es altamente personalizable
-Fácil manejo y sincronización de datos y contraseña
-Acelerador de descarga y hardware

Safari
Navegador web de la compañia Apple y casi el equivalente al Internet Explorer o Edge de Microsoft.
Por lo tanto, también se incluye en el paquete de software de los dispositivos portátiles de Apple con IOS
Caracteristicas:
-Apto para equipos de sobremesa, portátiles y dispositivos móviles
-Disponible para macOS e IOS
-Buena protección de datos porque Apple no registra ningún dato de los usuarios
-Optimizado para los dispositivos de Apple y el almacenamiento en ICloud

Microsoft Edge
Como resultado de las crecientes deficiencias en seguridad y de su reputación como navegador obsoleto,
los esfuerzos se concentraron en el desarrollo de Microsoft Edge, el sucesor de IE, que sería lanzado al mercado junto con
Windows 10 y también funcionan en Windows 7 y Windows 8
Caracteristicas:
-Apto para equipos de sobremesa, portátiles y dispositivos móviles
-Disponible para Windows, macOS X, Xbox One, Android e IOS
-El filtro de pantalla inteligente protege contra los ataques de malware y phising
-Resolución Full HD al reproducir videos
-Modo de lectura y lista de lectura

Opera
Opera existe desde 1996. Con más de 100 millones de instalaciones en todo el mundo,
Opera Mini es uno de los navegadores móviles más utilizados.
Además, el monopolio de los navegadores en las actuales consolas de Nintendo lo acapara Opera.
Caracteristicas:
-Apto para equipos de sobremesa, portátiles y dispositivos móviles
-Disponible para Windows, macOS, Linux y Android
-Borrado automático de la memoria caché
-Modo turbo para conexiones de internet lentas
-Una cómoda barra lateral muestra el Messengery los marcadores

Brave
Navegador web construido sobre el código abierto Chromium lo que hace similar a navegadores populares como Google Chrome.
Pero lo que distingue a Brave es su compromiso con la privacidad del usuario y la seguridad de los datos.
Brave está diseñado para priorizar la privacidad del usuario al bloquear rastreadores de sitios web,
anuncios intrusivos y otros elementos que comprometen tu seguridad en línea.
Caracteristicas:
-Bloqueo de anuncios y rastreadores
-Brave Shields
-Brave Rewards
-HTTPS Everywhere integrado

Vivaldi
Navegador multiplataforma desarrollado por Vivaldi Technologies, es el único que te permite la personalización
de prácticamente cualquier elemento de su interfaz, pudiendo colocar la barra de direcciones y pestañas
en la parte inferior, lateral izquierda o lateral derecha de la ventana si así lo deseas
Caracteristicas:
-Disponible para Windows, OS X, Linux, Android e IOS
-Rápido y fiable
-Sincronización de datos
-Buenas opciones para la expansión de la función

Tor Browser
Navegador web de código abierto cuyo objetivo es ofrecer a usuarios de internet una experiencia
de navegación absolutamente anónima, lo que hace una herramienta indispensable para periodistas
Caracteristicas:
-Encriptación compleja de datos antes de ser enviados por internet
-Desencriptación automática de datos en el lado del cliente
-Anonimidad total sin importar el servidor o la página web
-Permite visitar páginas bloqueadas en nuestra región
-Realiza tareas sin revelar la IP verdadera del usuario
-Es de código abierto, cualquier programador puede ver como funciona

Yandex Browser
Desarrollado en 2012 para dispositivos móviles y ordenadores por la corporación rusa
de búsqueda web que utiliza el motor de renderizado Blink utilizado en el proyecto de código abierto Chromium
Caracteristicas:
-Alta velocidad
-Integración con Yandex
-Funciona bien en PC débiles
-Consumo de memoria RAM reducido

Chromium
Navegador de código abierto que le permite modificar el código para ampliar las funcionalidades según sus necesidades.
Esto puede hacerse a través de la comunidad de desarrolladores de Google solo por los desarrolladores.
Al tratarse de un navegador de código abierto, también se pueden construir otros navegadores utilizando Chromium como código base.
Muchas aplicaciones utilizan el código fuente de Chromium en sus frameworks.
Caracteristicas:
-Actualizaciones y mejora frecuentes
-No realiza un seguimiento de datos
-No restringe a los usuarios a utilizar extensiones de Chrome Web Store
-Permite crear otros navegadores y aplicaciones utilizando su código

Front End VS Back End

FRONT END
Es la capa que se encuentra por encima del Back-End,
ya que comprende menús despegables, imágenes, iconos, colores, elementos gráficos, animaciones…….
Resumiendo, mejora la experiencia de navegación para el usuario
Para desarrollar el Front-End se necesita conocer idiomas esenciales como Javascript, CSS y HTML;
Y frameworks como Bootstrap o Angular entre otros.

BACK END
Es la capa de código al que los usuarios no tienen acceso,
está compuesto por uno o más lenguajes de programación empleados para el desarrollo de estructuras web.
El Back-End incluye información sobre como funciona un sitio en particular, sus cambios y sus actualizaciones.
Son aquellos aspectos que las personas no pueden ver en el navegador, entre ellos los servidores y bases de datos.
Los desarrolladores de back-end gestionan bases de datos, administran el contenido,
se encargan de la seguridad y crean la estructura de la página web

DIFERENCIAS
-Son conceptos diferentes
-Roles distintos: Front-End-->Aspectos visuales con los que experimentas. Back-end--> Facilita la interacción web
-Fundamentos esenciales:Lenguajes de Back-End-->Java,Python,PHP, etc Front-End--> HTML, CSS y Javascript.
